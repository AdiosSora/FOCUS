# FOCUS
カメラが識別した手のポーズ情報を用いてWindows端末を操作するアプリケーションです。<br>
ポインターの主な操作、独自スクリーンキーボードによる文字入力。<br>
Windwosショートカットなどを手の操作のみで実行が可能になります。
<h1>Requirements</h1>
・Start.batファイルを起動すると下記のライブラリが自動インストールされた後、起動します。
